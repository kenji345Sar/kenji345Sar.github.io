ツールガイド

中小企業がIT支援・業務改善で使うツールを、領域別に整理しています。 どのツールがどの業務に向き、どこで詰まりやすいかを、現場目線で解説します。 特定ツールの紹介ではなく、「自社にどれが合うか」を判断するための地図として使ってください。

各ツールは順次記事として公開予定です。

領域A：AI業務組み込み（生成AI / RAG / アシスタント）

  • Microsoft 365 Copilot 提供: Microsoft / 対象: 365 利用企業 / 用途: 文書作成・会議要約・データ分析の補助

    既存の Microsoft 365 環境に組み込まれる AI アシスタント。導入のしやすさと、現場で「使われない問題」のバランスが論点。

    準備中
  • Notion AI 提供: Notion / 対象: Notion 利用企業 / 用途: 社内ドキュメントの整理・検索・要約

    既存 Notion ワークスペース上で動く AI。社内ナレッジが Notion に集約されている前提で効くツール。

    準備中
  • Glean / 国産RAG SaaS 提供: Glean ほか / 対象: 中堅以上の情シス / 用途: 社内ドキュメント横断検索（RAG）

    社内に散らばる Slack・Drive・Confluence などを横断的に検索する RAG 系ツール。コストと費用対効果が論点。

    準備中

領域B：業務SaaS（情報整理・申請・コラボ）

  • kintone 提供: サイボウズ / 対象: 中小企業 / 用途: 業務アプリ・申請フロー

    中小企業に広く採用される業務アプリプラットフォーム。導入は簡単だが、連携・拡張で詰まる典型がある。

    準備中
  • Slack / Microsoft Teams 提供: Slack, Microsoft / 対象: 全業種 / 用途: 社内コミュニケーション

    情報のサイロ化を解く前提のチャット基盤。AI 連携・ワークフロー機能の使い分けが論点。

    準備中
  • Notion 提供: Notion / 対象: 全業種 / 用途: 社内ナレッジ・プロジェクト管理

    社内ナレッジ集約と業務オペレーションの中核として使えるドキュメント／DB ツール。

    準備中

領域C：業務自動化（連携・バッチ）

  • Make.com / Zapier 提供: Make, Zapier / 対象: 中小企業の業務担当 / 用途: SaaS間連携・自動化

    ノーコードでSaaS間の連携・自動化を組むツール。簡単な連携は素早く組めるが、複雑な業務ロジックは限界がある。

    準備中
  • Power Automate 提供: Microsoft / 対象: Microsoft 365 利用企業 / 用途: SaaS連携・RPA

    Microsoft 365 環境での業務自動化・RPA。デスクトップ版の RPA まで含めて検討対象になる。

    準備中

領域D：開発支援（情シス・開発担当向け）

  • GitHub Copilot 提供: GitHub / 対象: 開発・情シス / 用途: コーディング補助

    コード生成 AI として最も普及。法人導入時のセキュリティ・ライセンス論点を解説。

    準備中
  • Cursor 提供: Cursor / 対象: 開発・情シス / 用途: AI ファースト IDE

    AI と対話しながら開発する IDE。レガシーコードの調査・改修支援にも使える。

    準備中
