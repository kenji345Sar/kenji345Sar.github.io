業務改善・IT支援ができる個人の方へ — 案件に辿り着くルート整理

業務改善・IT支援ができる個人（社内SE経験者、業務システム改修経験者、要件整理が得意なエンジニア）が、 案件に辿り着くためのルートを整理しています。

現状、IT導入補助金・ミラサポ・Visasq・SaaSパートナー・商工会・DX支援機関などが点在しており、 SES のような「登録すれば案件が紹介される」単純構造が業務改善領域には存在しません。 結果、需要も供給もあるのにマッチングしないという機能不全が起きています。

このページは、その点在を可視化するためのものです。 各ルートの入口・要件・単価レンジ・現実をフラットに記載していきます。 私自身もこれらに登録し、その実体験を順次記事化していく予定です。

A. インバウンド型（登録 → 企業から声がかかる）

  • Visasq Lite / Visasq 提供: ビザスク / 対象: 経歴ベース / 形態: スポット顧問・1時間〜

    経歴を登録すると企業から声がかかる、スポット顧問プラットフォーム。 Visasq Lite は比較的緩い経歴審査で個人事業主も登録しやすい。 単価は1時間1〜3万円が中心、月顧問契約に発展するパターンもある。

    準備中
  • ProSharing 提供: サーキュレーション / 対象: 経歴ベース / 形態: 顧問・業務委託

    中堅以上のプロ人材マッチング。業務委託・顧問契約が中心で、業務改善・DX支援案件も流れる。

    準備中
  • MIMIR 提供: ミーミル / 対象: 経歴ベース / 形態: スポット相談・専門家マッチング

    経営課題に対する専門家マッチング。経歴審査はやや厳しめ。

    準備中

B. 公的登録型（公的窓口経由で相談が流れる）

  • IT導入補助金 登録IT導入支援事業者 所管: 中小企業庁 / 対象: ITツール提供 or 支援要件あり / 形態: 補助金経由の案件流入

    中小企業庁のIT導入補助金で、補助対象のITツールを提供する事業者として登録する制度。 個人事業主でも登録可能。登録すれば補助金経由の案件が定常的に流れる構造になる。 要件・申請手順を整理します。

    準備中
  • ミラサポplus 登録専門家 所管: 中小企業庁 / 対象: 専門領域あり / 形態: 公的相談・派遣

    中小企業向けの公的相談プラットフォームに専門家として登録。無料相談が中心で単価は低いが、信用付与・実績作りに有効。

    準備中
  • よろず支援拠点 登録専門家 所管: 中小企業庁・各都道府県 / 対象: 専門領域あり / 形態: 公的相談・派遣

    都道府県ごとの中小企業相談所。コーディネーターからの依頼を受ける形で活動。地域単位での登録になる。

    準備中

C. 副業・業務委託マッチング

  • BizReach系（GLIT / ProSharing 等） 提供: ビズリーチ系列 / 対象: ハイクラス / 形態: 業務委託・顧問・副業

    ハイクラス向けのプロフィール登録型マッチング。エージェント経由よりも本人ブランドが前面に出る。

    準備中
  • クラウドリンクス / Pasona JOB HUB Friend / 複業クラウド 対象: 副業・業務委託前提 / 形態: マッチング

    副業・業務委託のマッチングサービス。業務改善・IT支援系も流れる。プロフィールの質次第。

    準備中
  • 業務委託エージェント 対象: 即戦力エンジニア / 形態: SES / 業務委託

    SES寄りのエージェント。下請け案件が中心で、個人ブランドは立ちにくい。ランウェイ確保の保険として位置付けるのが現実的。

    準備中

D. SaaSパートナー認定（ツールベンダー経由の案件流入）

  • Microsoft Copilot / Microsoft 365 パートナー 提供: Microsoft / 対象: 認定要件あり / 形態: パートナー経由の案件流入

    Microsoft Copilot の法人導入支援を行うパートナー認定。要件・売上ボーダーを整理します。

    準備中
  • kintone パートナー 提供: サイボウズ / 対象: 認定要件あり / 形態: パートナー経由の案件流入

    kintone を導入する中小企業からパートナー経由で案件が流入する。要件・契約形態を整理します。

    準備中
  • Notion / Slack 法人パートナー 提供: Notion, Slack / 対象: 認定要件あり / 形態: パートナー経由の案件流入

    法人導入を支援するパートナー登録。海外SaaSのため日本法人の有無で要件が変わる。

    準備中

E. その他（自前構築）

  • HP・X・LinkedIn からの直接インバウンド 対象: 自前 / 形態: 自分のサイト・SNS経由

    自分のサイトと SNS で発信を続け、検索・SNS経由で問い合わせを受ける形。 立ち上がりまで1〜2年かかる前提だが、最も自分のブランドが残る。 このサイト自体がそのアセット。

    本サイトが該当
